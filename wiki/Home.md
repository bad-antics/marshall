## Overview
Marshall is a privacy-focused web browser built with Rust and GTK4, designed for security researchers and privacy enthusiasts.

## Features
- WebKitGTK6 rendering engine
- AI Assistant integration
- Dark theme interface
- Tab management
- Bookmark support
