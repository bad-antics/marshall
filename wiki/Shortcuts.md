# Keyboard Shortcuts

## Navigation
| Shortcut | Action |
|----------|--------|
| `Ctrl+L` | Focus URL bar |
| `Alt+Left` | Go back |
| `Alt+Right` | Go forward |
| `F5` / `Ctrl+R` | Reload page |
| `Escape` | Stop loading |

## Tabs
| Shortcut | Action |
|----------|--------|
| `Ctrl+T` | New tab |
| `Ctrl+W` | Close tab |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Ctrl+1-9` | Switch to tab 1-9 |

## Window
| Shortcut | Action |
|----------|--------|
| `F11` | Toggle fullscreen |
| `Ctrl+Q` | Quit |
| `Ctrl+N` | New window |
| `Ctrl+Shift+N` | New private window |

## Features
| Shortcut | Action |
|----------|--------|
| `Ctrl+D` | Add bookmark |
| `Ctrl+B` | Show bookmarks |
| `Ctrl+H` | Show history |
| `Ctrl+Shift+I` | Developer tools |
| `Ctrl+F` | Find in page |

## AI Assistant
| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+A` | Toggle AI panel |
| `Ctrl+Enter` | Send to AI |
