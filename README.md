- Rust 1.70+
- GTK 3.0
- WebKitGTK

### Debian/Ubuntu
\`\`\`bash
sudo apt install libgtk-3-dev libwebkit2gtk-4.0-dev
\`\`\`

### Arch
\`\`\`bash
sudo pacman -S gtk3 webkit2gtk
\`\`\`
