# Changelog

All notable changes to Marshall Browser will be documented in this file.

## [2.1.0] - 2026-01-24

### Added
- **Workforce Center** - Complete employee management system
  - Employee tracking with roles and hourly rates
  - Time card management with clock in/out
  - Project tracking and assignments
  - Payout dashboard for payroll management

- **VoIP Center** - Secure communications hub
  - Full-featured dialpad interface
  - Call display with duration tracking
  - Contacts management system
  - Recent calls history
  - SIP settings configuration

- **AI Model Selection** - Enhanced Dr. Marshall configuration
  - Support for multiple AI providers (Anthropic, OpenAI, Google, Mistral, Local)
  - Configurable model parameters (temperature, max tokens)
  - System prompt customization
  - Local model support via Ollama

### Enhanced
- Menu system with new Workforce and VoIP links
- Dr. Marshall AI assistant with improved responses
- Overall UI consistency and polish

## [2.0.0] - 2026-01-20

### Added
- Initial GTK4/WebKit2GTK implementation
- Privacy-focused browsing with Tor integration
- Built-in OSINT tools
- Password manager
- NullSec Hub integration
- Internal page routing (marshall://)

## [1.0.0] - 2026-01-15

### Added
- Initial release
- Basic browser functionality
- Privacy features
